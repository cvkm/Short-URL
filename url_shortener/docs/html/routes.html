routes module¶

All the routes are defined

routes.add_link()¶

Retrieves the Original URL from user, validates whether the URL is valid or not, converts it into Short URL and stores both in the database. Loads the add link page displaying both the Original URL and the Short URL

routes.auth()¶

Loads the login page and validates the responses of the user. Upon successful validation the user is redirected to the logs page

routes.before_request()¶

Checks for user in session

routes.index()¶

Loads the Home page and stores the logs of visitor

routes.page_not_found(e)¶

Throws a 404 error, for page not found.

routes.redirect_to_url(Short_URL)¶

Redirects to Original URL upon accessing Short URL and increases the visits of the Short URL by one in the database

Parameters

Short_URL (string) – Six random characters

routes.stats()¶

Checks whether the user in session has access to the logs. If user has access, then it loads the logs page dispalying the stats related to Links else the user is redirected to the login page
